# Migration from Vue 2 to Vue 3: Beta
Now it's work only for .js files
In the future it will support .ts, single-file-components

## Install
Use -g flag!
```
npm i -g vue2-to-3
```

### Example
```
migrate ./src/components/HelloWorld.js
```
